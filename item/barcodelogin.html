BARCODE LOGIN

Project details

Barcode scanner to login for 4659 Warrior Robotix attendance tracking and is made using zxing scanner. The release version has user admin authentication to prevent unauthorized sign-ins. Attendance information is sent using POST to a Ruby controller where the actual website attendence tracking is handled.

The barcode login scanner was requested by an admin of the robotics club, but it is not used in the actual attendance tracking due to poor internet connectivity leading to inefficient sign-in times. The barcode login is more of a "cool" way to check in for attendance. However, It was still a good introduction to the android api and mobile development.
